Important FAQ

What is Document Object Model(DOM)?

DOM stands for Document Object Model is a programming interface which represents the structure of a webpage. With DOM, we can manipulate a webpage such as modify or replace the text,images and other elements.

How do you select an element from the DOM?

There are a few ways to select an element or multiple elements in DOM. The first one is the most used "GetElementByID" which needs an ID to select an element and another almost similar one is querySelecter. For multiple elements to select Javascript provides a couple of methods such as GetElementsbyClassNames and querySelectorAll and GetElementsbyTagName. It's all used in different cases

What is event delegation in the context of a DOM and why is it useful?

Event delegation is a pattern where we delegate the event listener to a parent element to avoid attaching multiple event listeners to an individual element

How do you manipulate an element's attributes and styles using the DOM?

We can use different methods and properties to manipulate an element's attributes and styles.

    To Manipulate attributes

  • getAttribute(): Retrieves the value of an attribute.
  • setAttribute(): Sets or changes the value of an attribute.
  • removeAttribute(): Removes an attribute.
